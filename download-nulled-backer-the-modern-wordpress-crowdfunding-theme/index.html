What version of the plugin do i need?

Are you a single entrepreneur or organization wanting to raise money for your own projects?

Then you need the basic version of the plugin, or the free one. You can easily create unlimited projects, style them however you wish, structure them in tabs or dedicated pages, write your own content, etc.. You can also raise money for a single project using this plugin. Even if the theme showcases a crowdfunding website, you can use it for a single project in a landing page or whatever you think of.

The plugins are created by you (the administrator) or by other users that you delegate to do this via the WordPress Dashboard, just like any other post or page you create in WordPress. Watch the online video or look in the screenshots to see how the project builder looks like.

The main difference between the free plugin and the basic payed one is that the free one doesn’t allow for users to pledge on your projects. You would have to think of a different way to collect money and add the orders automatically via the IgnitionDeck Dashboard.

So whether you’re an artist, musician, entrepreneur, charity organization, or a simple person trying to raise money for that one dream project, this theme is for you and you can use it along with the free or base plugin.

Do you want to have crowdfunding website where users would be able to create profiles and add their own projects?

Then you need to use the Enterprise version of the plugin. With it, you can enable the front end dashboard and project submission area, so that users can register on your site and post projects without having access to your WordPress installation. They can setup individual PayPal address so each user will get his own money with no extra fees.
