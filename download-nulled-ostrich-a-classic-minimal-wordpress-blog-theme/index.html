Ostrich have 7 predefine template those are: 1. About Page Template This template can work like About us/About me/About author or About your team page. If you select this template then following options will be show in meta box. 2. Contact page This template define contact page. This template contain a Location Map, a details and a contact form. This template must require following info from option panel. 3. Full Width Page This template define full width page. If you assign this page then no sidebar will show in this page. And sidebar show hide option will not work if you select full width page. 4. Masonry Page This template define masonry blogs template. 5. Multiple Author This template define multiple author blog list. All author blog will be show here with all informations. 6. Ostrich Blog This template define a blog template with featured blogs list option. 7. This template define blogs template with left sidebar.

Page Sidebar Settings

Ostrich contain sidebar settings in page meta box. You can change sidebar positions left sidebar right sidebar even you can make full width layout without sidebar.

Featured Post

You can make any post featured by this option. Featured post will show ostrich blog page top carousel. Featured post is always off by default. If you want to make any post featured then you have to on featured post option.
