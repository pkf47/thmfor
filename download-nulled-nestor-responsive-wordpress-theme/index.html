Download nulled Nestor - Responsive WordPress Theme


UPDATE 2014.10.12 – Version 1.1.1 Please update your theme.

Download theme

Nestor is a clean, modern and multi-purpose WordPress theme. The theme uses the powerful Twitter’s Bootstrap 3 and takes advantage of its features: grid system, typography, elements and more. It is fully responsive and looks great in smaller devices.

The theme was built with a lot of care and every part of the theme is very well explained and documented, so your life will be very easy working with it. Whether you are building a corporate, a designer’s portfolio or a photographer website we are sure that Nestor will perfectly fit your needs.

Other Versions

Features:

Supports One Page or Multi-Page Website Layouts HTML5 and CSS3 Bootstrap v3.1 Parallax Ready Ionicons Iconic Font Fully Responsive Design FlexSlider 2 Filterable MixItUp Portfolio Our team, Tabs, Accordions and more. Very good documentation Very clean and organized code! Supports all major browsers Dedicated Support

WordPress Theme Features:

Page Builder Custom per thumbnail cropping. Easily change theme primary colors from admin theme options page. Easily choose Google Fonts (with font preview) in theme admin options page. 1-click demo content import. Shortcode editor. Slider/Image/Video formats supported for Posts and Projects. Drag and drop gallery management. Localization ready, pot file included. Built on a full OOP , modular, load-on-demand, WordPress framework. Network install compatible. Supports child themes. Developed with WP_DEBUG and Debogger enabled.
