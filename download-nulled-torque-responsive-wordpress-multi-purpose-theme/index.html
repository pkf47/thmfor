Details & Features

Torque is a WordPress theme with a super clean user interface perfect for any corporate business, portfolio, creative, or ecommerce shop website. Packed with premium features such as the revolution slider, accordion wordpress slider plugin, visual composer page builder and much more, Torque is a super value. Make sure to check out each of the included 5 homepage styles, each has a completely different look and are all included with your purchase (plus you can use any of these elements to create your own homepage)!
