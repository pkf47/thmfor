RETINA READY

Creatika offers a great visual experience on high density screens due to the retina ready custom icon font integration and heavy CSS3 usage.

DRAG AND DROP PAGE BUILDER

The theme integrates Visual Composer for WordPress which is a drag and drop content builder that allows you to build any layout you can imagine – no programming knowledge required.

22 CUSTOM VISUAL COMPOSER ELEMENTS

Visual composer is very flexible out of the box with 40+ predefined elements but we wanted to take it to the next level and added 22 custom elements of our own. The goal was to make good on our promise that no programming knowledge is required to work with this theme.

8 CUSTOM WIDGETS

You can add content to the widgetable areas much easier using the 8 custom widgets that come with the theme: Flickr widget, Social media widget, Twitter, Call to action widget, Contact Info, Navigation, Latest post widget and Contact form 7 widget.

CUSTOM ICON FONT INTEGRATION

The theme has a selection of retina ready icons included in our content box elements giving you a wide variety of options to choose from.

SIMPLE COLOR SCHEME CUSTOMIZATION

Changing the color scheme is as simple as selecting some accent colors, the theme takes care of the rest.

POWERFULL THEME OPTIONS PANEL

The theme has a powerful theme options panel with a lot of options. It allows you to make the design boxed/ wide, change the background, change the typography, change the color scheme, make the header sticky, configure the footer to have between 1 to 4 widgetable areas per zone and set different sizes to each of them.

PARALLAX SECTIONS

Create an amazing parallax effect with this unique styling and show your clients what you are all about!

ADVANCED CONTACT FORMS

Creatika supports the Contact Form 7 plugin that allows you to easily create advanced forms.

REVOLUTION SLIDER

The theme integrates Revolution slider as well. This plugin features tons of unique transition effects, an image preloader, video embedding, autoplay that stops on user interaction and lots of easy to set options to create your own effects.

UNLIMITED SIDEBARS

Creatika has a unique sidebar and page layout manager. Easily create and manage an unlimited number of sidebars and assign them on a per page basis.

SIMPLE INSTALATION AND SETUP

Quickly install the theme through WordPress or FTP and then just import the demo content provided, add widgets and set up the menu and you are done. It doesn’t get much simpler then this!

TRANSLATION READY

Easily translate the theme using the Codestyling Localization plugin.

BONUS: 49 FULLY LAYERED PSD FILES

If you ever need to change / extend the design we provide 49 fully layered psd files that you can use for quick mock-ups. These are the same quality you would find in an item in the PSD section.

EXTENSIVE DOCUMENTATION

We wrote the documentation with beginner WordPress users in mind. It is written in a step by step format and takes you through setting the theme up and customizing it. It is divided into sections and sub-sections so power users can just skip to what they are interested in.

5 STAR SUPPORT 24/7
