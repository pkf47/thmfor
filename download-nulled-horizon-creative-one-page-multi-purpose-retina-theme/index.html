Horizon is a modern and elegant multipurpose retina and responsive one page WordPress theme. Hand crafted to a fine detail with beauty and ease of use in mind. Perfect for any creative, business, photography, personal or music websites. Make Horizon your own.

Features

One click installer – With Horizon’s demo installer, you’ll be set up and ready to customize your site in seconds

Parallax – Make any section background or the slider parallax

Language Support

RTL Support WPML Compatible – for multilingual support Texts editable via option panel – Change to your language without the hassle of WPML. (some blog texts need to be translated the regular way.

Revolution Slider ($18 value) – This is a very flexible and powerful slider

Simple built in slider – Want to keep things quick and simple? With the build in slider you can get going quickly and if you want you can change the colors, fonts and animation types

Visual Composer Page Builder ($28 value) – Drag and drop page builder. All of horizon’s shortcodes can be added and edited via visual composer

Page builder is optional – If don’t want to use a page builder, you can still add shortcodes via a editor menu

Portfolio – Filterable and smoothly animated ajax portfolio

Woocommerce ready – Want a shop? No problem, Woocommerce is now supported

Responsive – Resizes on mobile devices

Retina – Looks sharp on retina devices

Post Formats – Standard, Aside, Gallery (slider), Quote, Link, Video and Audio post type options

Unlimited colors choose your color with a color picker or enter a hex value. You can pick any color you like!

Google fonts – You can choose a font from the Google set for your typography. There’s hundreds to choose from.

Child theme – Theres a child theme boiler plate included, get modifying quickly.

Font awesome icons – You can use any icon from the font awesome pack, they are scalable and look great on retina

If you are interested in a particular feature you don’t see listed, support is happy to answer pre-sale questions via the comments
