Overview

Snaps can be used as multi-purpose portfolio and business WordPress theme or for any other usage. Feel free to create your own website with this finished template. Containing more than fifteen different layouts. Snaps WordPress theme can be used for multi-purpose websites. Wether you want to create a portfolio, a business website, a blog or even a personal profile page, Snaps is always the right solution. Use Theme Options and Shortcodes to customize your website. The userfriendly WordPress interface will help and guide you through to build a fully working and clean website.

Download theme

Snaps has a clean and minimalistic design which helps you to create an awesome and powerful project. The whole template was built based on the latest standards and recommendations. Snaps theme is powered by Roots (WordPress starter theme), Vafpress (WordPress framework) and Twitter Bootstrap v3 (the most popular front-end framework). It means that it will be easier for you to do the changes and to increase its possibilities, thanks to many plugins it possesses.
