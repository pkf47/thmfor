Vilan is a professional business, eCommerce and forum theme for WordPress built with Bootstrap and powered by Visual Composer. The theme design is flat and simple with vibrant color and custom modules specially created for easy content creation. The primary colors used are blue and dark grey specially chosen because of their visual influence on the user and their contribution to a better browsing and shopping experience. The shop module is powered by the well known WooCommerce plugin and the forum module is powered by the famous bbPress.

Take your time and see bellow some of the theme characteristics

Ecommerce features

The shop feature is powered by the most popular and powerfull ecommerce plugin available – WooCommerce. A very flexible plugin which gives you the possibility to sell anything, from digital downloads to women shoes. You can find a detailed description of its capabilities . Some of the features this theme has to offer are: - Full width and sidebar layout feature for the products page - Unlimited product variations with different prices for each variation - Catalog mode (turns the buying feature off and you can use the theme like a catalog) - Possibility to show/hide: Product title, price, add to cart, category, description, product meta, tabs, upsells, cross sells, related products. - Shortcodes for sale products and featured products

Demo conten and theme documentation

To speed up the installing process you have available demo content as an xml file which contains pages, posts, products and post types that illustrates the theme features. Also with the ease of “1 click install” you have the option to load the theme settings, sidebars (widgets), menus, woocommerce and bbpress plugin settings (Kudos ). Other features
