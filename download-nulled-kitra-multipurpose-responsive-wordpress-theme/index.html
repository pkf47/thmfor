Kitra is a fully responsive premium Wordpress theme that suits to every concept. It comes with 5 different concepts which are One Page, MultiPage, Restaurant, Photography and Wedding. Kitra is ready to create your own concept with powerful page builder in it without touching any code. It includes Premium Visual Composer page builder and Premium Revolution Slider. It comes with special restaurant menu pages and awesome one page portfolio sliders in it. Endless possibilities for your projects.

It is well documented and has 3 header options. It has powerful admin panel which you can change all typography and colors of the theme with an ease.
