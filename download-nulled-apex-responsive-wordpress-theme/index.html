Apex Responsive WordPress Theme 1.0.11

Apex Responsive WordPress Theme offers a unique intuitive interface, featuring the trendy one page layout design, and 6 built in colour themes. You can also easily customize colours and styling with numerous theme and page settings. Built on Bootstrap 3, Apex has been carefully crafted to offer a reliable and robust theme for your project. Also included is a separate blog section, which has a blog archive, single blog post pages and supports 5 different blog post types – text, single image post, multiple image post, video post and quote. This theme template includes team members, portfolio, services, client logo, contact, blog page and more.
