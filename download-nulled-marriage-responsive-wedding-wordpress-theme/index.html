How to update the theme from older versions

Replace the files of the theme from your server. If you don’t made modifications to the old file you have. Save/Store all the custom settings added in the custom admin panel. Just those custom values added in that panel only. Not the post/pages content. Remember there is no automatically option to save those values, you will need to copy/paste them somewhere and insert them again after Reseting the admin panel.
