Download nulled Horizillax - Horizontal Parallax Wordpress Theme


Horizillax is a unique horizontal parallax based single page, responsive, wordPress theme. Horizillax can be used for any business, corporate, portfolio, products, marketing, etc. This theme is perfect for creative designers, artists, photographers. It is very easy to setup and use (just 4 simple steps).
