Download nulled Skoty - Responsive Multipurpose Blogging Theme


Briefly

Download theme

Skoty is much more then just a creative blogging theme. It contains 3 plugins that will help you to elevate your blog to a new level by allowing you to build pages with various layouts, letting your visitors show their appreciation by liking your posts and staying socially connected.

Features

Responsive & Mobile Friendly Retina display ready WordPress 3.9 ready Multilingual Ready WPML Compatible Content Builder Plugin: BoxBuilder by atticthemes A Portfolio Plugin: Attic Portfolio by atticthemes View Counter & Likes System Plugin: Blog Extender by atticthemes Custom Built WordPress Customizer Options Archives Page Template Custom Made Responsive jQuery Lightbox plugin Header Image Support with Parallax Effect Masonry Galleries Blog Post Sharing (Google Plus, Facebook, Twitter, Pinterest) 30+ Icons in a Custom Built Social Icons Plugin by atticthems Contact Form 7 Support Smooth Scrolling for Google Chrome (SmoothScroll v0.9.9) HTML5 and CSS3 Localization Support FontAwesome Iconic Font Commented Documentation Videos
