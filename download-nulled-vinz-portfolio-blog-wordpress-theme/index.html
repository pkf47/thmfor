Vinz is a modern and clean designed Portfolio, Photography and Blog Theme, based on a 100% responsive grid. It comes with the easy to use Drag & Drop Page Builder . ### See what you get

Features Page Build Visual Composer
All Vinz exlusive features are integrated to the Page Builder. No need to remember any shortcode. Full OnePage Support
You can run Vinz as an OnePage. Just one click in the Theme Options to enable it. Portfolio & Team
Create your projects / members and place them wherever you want through the Page Builder. Powerful Load More Function
Pages, even the Blog Page, can be placed on any other page. ****
Load More on Click or on Scroll, it’s your choice. Horizontal Timelines
Create Timelines easily through the Page Builder Grid-A-Licious Blog
Blog Posts in a Grid, 100% Responsive. Blog comes with a Load More Function. Ajaxified Comments
No page reload required for posting a new comment. A lot Theme Options
If fonts or colors, preloader image or which blog informations (categories/author/tags…) to show, you can change it through Theme Options. One-Click Demo Data
Only one click to get the same page as our preview. You can choose between Classic or Onepage Demo Data. 1000+ Google Fonts and 350+ Icons by Font Awesome
You can decide which fonts you want to use. Goole Maps Api Editor
It’s super easy to create a styled map with multiple markers. Contact Form
Small & Simple Contact Form integrated. Automatic Updates
