Download nulled MongKok - Modern Onepage Resume / Personal Theme


Impress your future employers with this beautifully crafted resume theme!

Download theme

MongKok is a modern onepage resume / personal WordPress theme built on Bootstrap 3 with lots of options for customization. It features a resume section with clean typography and animated charts, a portfolio section and a book section. With the code snippets page that comes with the template, it is easy to create your own contents that are consistent with the template in terms of style.

PHP 5.3+ is required for this theme to work properly. For the live preview on mobile devices, please click .

HTML version is included in the theme package. Please remember to give this template a rating, and subscribe to get notified of updates in the Downloads tab of your themeforest account. Thank you very much for your feedback.

Latest version: 1.1.2 (Feb 17, 2015, )

Testimonials Theme Support

Theme support covers bug fixing and resolving problems when setting up or using the theme, but does not cover adapting or fixing the theme to support any third-party plugin that is not used by the theme.

Key Features Modern design Responsive layouts Flexible color scheme (single color / dual colors) 5 predefined color schemes 4 navigation bar positions (top, bottom, left, right) 6 CSS3 Off-canvas navigation animations (push, reveal, scale up, rotate in, rotate out, rotate & reveal) Resume section with clean typography 4 types of charts (bar chart, pie chart, donut chart, progress chart) with on-scroll animations AJAX portfolio section 3 types of predefined layouts plus the option to use your own layout for portfolio details Image slider and YouTube / Vimeo videos can be used in portfolio details Book section with CSS3 animated books 2 predefined 3D transitions for animated books plus 2D fallback Gallery section with lightboxes for images, Youtube / Vimeo videos and webpages Styled / original Google Maps Working AJAX contact form ** included**

Lots of code snippets to help you create your own content
