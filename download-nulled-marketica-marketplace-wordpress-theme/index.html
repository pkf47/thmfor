Download nulled Marketica - Marketplace WordPress Theme


Marketica v2.0 is the best marketplace theme for WordPress and WooCommerce. Period!

Download theme

Marketica v2.0 allows you to turn your store into a vendor marketplace. You can assign existing products to vendors, or they can add and edit their own ones. Every vendor will have a dedicated vendor page on your site where they will be able to list all of their products, but they will also show up in your normal shop pages so you can sell your own products alongside those of your vendors.

We have prepared 4 demos for you!

DEMO #01: Standard WooCommerce

This demo is for everyone who want to create a standard WooCommerce shop without any vendor marketplace plugin.

DEMO #02: WC Vendors

This demo is for everyone who want to create a marketplace using WC Vendors plugin. This is a free plugin that can be downloaded

DEMO #03: WooCommerce Product Vendors (WooThemes)

This demo is for everyone who want to create a marketplace using WooCommerce Product Vendors plugin. This is a premium plugin that need to be purchased separately .

DEMO #04: Socio Multi Vendors

This demo is for everyone who want to create a marketplace using Socio Multi Vendors plugin. This is a premium plugin that need to be purchased separately .

COMPATIBILITY:

WordPress 4.1+ (self-hosted) WooCommerce 2.2, 2.3
