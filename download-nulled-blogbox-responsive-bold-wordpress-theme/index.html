Download nulled BlogBox - Responsive & Bold WordPress Theme


BlogBox is a minimal, bold, responsive and easy to use photoblog-style WordPress theme. Its a perfect fit for artists, photographers, or just creative people looking for a theme with clean design to share their photos, galleries, videos and articles.

Download theme

Ready for Mobile Devices

Responsive design adapts smoothly to different screen sizes, so your readers can access your blog from any device with a web browser.

Post Formats

BlogBox supports link and quote post formats, so you can share your favorite links/quotes with style. The quote post format has a unique feature – you can add a background image for you quote.

The following is a list of main theme features:

Translation file included – use BlogBox in any language Responsive layout – your blog will be accessible on various devices (iPhone, iPad, Android, etc.) Retina display ready – all icons are optimized for retina devices WordPress post formats – specific styles for link and quote posts Theme customizer Social media icons Author box support (useful for multi authors) Easy setup with easy to follow instructions

Supports JetPack Plugin

With JetPack plugin you will have many awesome features like infinite scroll, tiled gallery and more.
