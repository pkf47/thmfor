The Personalist WordPress theme is a custom design minimal personal blogging, portfolio theme for displaying artwork. The Personalist WordPress theme features a fully responsive framework that looks fantastic on any mobile device and retina display ready for high-quality resolution graphics. This template suitable for personal bloggers, photographers, designers, graphic designers, showcase sites small business sites etc.

Download theme

It contains custom blog layout and custom portfolio template (with a super awesome jquery plugin) layout options and custom page template examples.
