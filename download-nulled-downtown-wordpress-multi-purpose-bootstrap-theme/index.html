Download nulled Downtown - WordPress Multi-Purpose Bootstrap Theme

Downtown is fully responsive clean and modern WordPress theme that can be used for any kind of page. Theme is based on Bootstrap 3 front-end framework and comes with tons of useful shortcodes. Unlimited portfolios and galleries, standard and grid styles, parallax sections, fixed header, gallery albums and many, many more features to check out. Theme was also tested with plugins like WPML and qTranslate and is fully translatable to other languages. All of this and more with awesome and quick support that will help you in case of any problems.
