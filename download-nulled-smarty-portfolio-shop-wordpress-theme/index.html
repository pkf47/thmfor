Smarty is a profesional business, portfolio and eCommerce theme for WordPress built with Bootstrap and powered by Visual Composer. The theme design is flat and simple with vibrant colors and custom modules specially created for easy content management. The primary colors used are green, blue and grey specially chosen because of their visual influence on the user and their contribution to a better browsing and shopping experience. The shop module is powered by the well known WooCommerce plugin.

Take your time and see bellow the main characteristics of the theme

Ecommerce features

The shop feature is powered by the most popular and powerful ecommerce plugin available – WooCommerce. A very flexible plugin which gives you the possibility to sell anything, from digital downloads to women shoes. You can find a detailed description of its capabilities . Some of the features this theme has to offer are: - Full width and sidebar layout feature for the products page - Unlimited product variations with different prices for each variation - Catalog mode (turns the buying feature off and you can use the theme like a catalog) - Possibility to show/hide: Product title, price, add to cart, category, description, product meta, tabs, upsells, cross sells, related products.

Demo content and theme documentation

To speed up the installation process you have available demo content as an xml file which contains pages, posts, products and post types that illustrates the theme features. Also with the ease of “1 click install” you have the option to load the theme settings, sidebars (widgets), menus and woocommerce plugin settings (Kudos ). Other features

Import/export feature to backup your theme settings. Unlimited sidebars (with the help of woosidebars) PSD available Child theme support SEO optimized Valid HTML Build in contact form
