About Readme

Readme is a responsive WordPress theme focused on readability with a minimalist design and optimized for mobile.

Download theme

Features

Minimalistic design focused on readability Optimal Reading Experience On Any Device Beautiful Typography & Distraction Free Reading Responsive Design & Retina ready HTML5, CSS3, jQuery powered Flexible Images & Videos 5 blog layouts (with or without sidebar versions) Complete blog system (all post formats supported) Unlimited Portfolios (Muliple portfolio page support) Filtered portfolio with cool css3 animations Responsive mobile optimized image gallery Self hosted html5 audio/video (one .mp4 file for all browsers!) Responsive video embed in various aspect ratios (youtube/vimeo …) Cross browser styled form elements 30+ social media icons Unlimited Sidebar Custom Widgets (Flickr, Instagram, Pinterest, Dribbble, Picasa, YouTube) Footer widget locations (4 columns) Custom Settings Panel Theme Customizer Support 600+ Google Fonts Ajax contact form with validation SEO & Google Analytics ready Custom Shortcodes Translation Ready (.po files included) Extensive Documentation &
