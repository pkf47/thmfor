Update instructions

**If you are updating to Realocation 1.2 from previous version, in order to preserve compatibility you need to follow this steps ** ( if you want to keep your changes to filters and widgets )

1. Update the theme, and the plugins - update of plugins will be requested in administration, you just need to confirm it, after updating theme folder. 2. Import exports - exports/hydra-1.2.xml - new Filters will appear - Filter Horizontal + Filter Vertical 3. Head to Appearance -> Widgets, and resave any map widget ( otherwise the map won’t work ), select either Horizontal or Vertical filter - which one you want. 4. Enable Vertical Filter in the primary sidebar, instead of old Filter ( disable old filter )
