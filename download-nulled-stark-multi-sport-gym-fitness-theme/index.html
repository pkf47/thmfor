Visual Composer – Drag & Drop Page builder

Stark includes a drag and drop layout builder. With this builder you can easily built pages, portfolios and posts. It’s main feature is that you can nest elements. So you can add one into another. You can create unique layouts with over 60+ shortcodes. The layouts shown on our demo site are just a few examples how it can be build.

There are no limitations for the created layouts. Layouts in this preview site show only a few examples of what you can make. Create as many as you want, style them as you want and present content which you want. Couldn’t be easier, right?

SEO optimization

Stark Wordpress Theme is supported for all mayor and great plugins for the best optimization you can have, like are “Yoast’s WordPress SEO”, “All in One SEO Pack” and others.

WPML – The WordPress Multilingual Plugin

WPML makes it easy to build multilingual sites and run them. It’s powerful enough for corporate sites, yet simple for blogs. With WPML you can translate pages, posts, custom types, taxonomy, menus and even the theme’s texts.

Theme options

Admin panel for Stark wordpress theme comes with easy to use Theme Options. This options offer you to change colors, fonts, layouts, some default values and much more with just clicks. This way you can adjust the entire page to suit you to perfection. All options are formatted into groups so there are very quick to locate.

Unlimited colors

With changing colors in Theme Options you may set your site to a color combination that suits you the best. Colors are not predesigned, so you may set any color you want.

All shortcodes have options to change them also. So you’re just 2 clicks away from getting your personal best color sheme.

Responsive design

Stark Wordpress Theme has a fully responsive design. This makes an optimal view for the content on any device using any resolution. You may check the responsive layout with changing your browser size.
