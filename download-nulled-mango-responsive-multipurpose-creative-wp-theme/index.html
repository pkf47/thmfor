Download nulled Mango - Responsive Multipurpose/creative WP Theme


Mango is a stylish, creative, multi-purpose, responsive and retina-ready WordPress theme. It features multiple home page types: fullscreen video, fullscreen slideshow and corporate. It’s suitable for small businesses, creative agencies freelance designers and photographers. Loaded with many powerful features like theme options, shortcodes, home content builder, customization options and much more.

Download theme

v1.0.2 – WP 4.0+ READY / WooCommerce Now Supported
Theme Features Fully responsive Design – Beautifully adapts to different screen sizes starting from small mobile phones to HD displays. Touch devices friendly. Retina ready – Enjoy ultra sharp graphics on hi-res displays. Highly customizable:- Customize logo, favicon and apple touch devices icon. Customizable starter skin (light/dark) and secondary color (color picker). Insert custom CSS for more styling control if needed. Use custom @font-face font for heading (Google free font or others). Filterable and/or paginated portfolio items +30 Shortcode with short code generator. A unique homepage content builder – Add, remove edit and arrange content sections. Multiple page templates:- 3 home page templates: Fullscreen YT video, fullscreen image slideshow and corporate. 3 portfolio page layouts: 3 columns, 2 columns and 2columns with sidebar. All with masonry layout ability. 2 blog page templates: normal layout and 2 columns masonry layout, both with sidebar. Contact page template with Google map location, contact info and working contact form with validation and basic security. About page template with team members gallery. Archives page template FAQs page template with 2 different layouts: normal layout and accordion layout. Multiple powerful custom post types:- Portfolio post type – Create portfolio items with ease and lot of options. Slides post type – Create Slider items with ease and lot of options. Work team post type – Create team members with ease and lot of options. Clients post type – Create clients thumbnails with ease. FAQs post type – Create FAQs with ease.

Powerful & uncluttered theme options panel. - Custom widgets:- Twitter feeds. - Social links. - Contact info. - Flickr photostream. - Instagram images. - Video embed. - Posts with thumbnails. - Recent work (portfolio). - Multiple sidebars (General, single post, page, portfolio and FAQs) - upto 4 widgetized header areas + 4 widgetized footer areas. - Blog post formats support (8 post formats). - Google Analytics code insertion. - Localization ready. - Support & updates
