ReadPress – WordPress Magazine Theme

ReadPress is a magazine WordPress theme. It is suitable for online magazine, news, or blog website. ReadPress is 100% responsive design and based on well-known CSS framework, Bootstrap. It is very easy to set up, the xml demo content is included in the theme package, and it is powered with Redux framework which offers an advanced theme options panel.
