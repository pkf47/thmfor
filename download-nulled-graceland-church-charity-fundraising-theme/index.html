Download nulled Graceland - Church, Charity & Fundraising Theme


Version 2.1 is available from 20.03.2014

Graceland is an excellent Church & Charity WordPress theme based on responsive Bootstrap 3.1. In Graceland you can create PayPal Donation campaigns, Events, Sermons, Galleries, online Shop with WooCommerce. Graceland is based on LiveComposer pagebuilder.

Using Graceland you can create Environmental, Church & Non-profit websites with Paypal Donation system. It’s powered by LiveComposer page builder – no coding required to build the pages you need. Please note that images are not included in PSD files.
