Download nulled Moje - vCard Bootstrap Responsive WordPress Theme


Moje is a stylish vCard theme loaded with features which makes it a perfect choice for your resume, minimalistic or even full blown portfolio website. You can either use it as a single page site or multi-page site. It is powered by Bootstrap 3 hence it works from desktops to tablets to smartphones flawlessly and awesomely.

Download theme

Moje is a feature rich WordPress theme. Some of the sailent features of Moje theme are:

Responsive design Retina ready theme WordPress 3.5+ compatible Single and multiple page layouts Dark and Light schemes 6 Colors options Custom shortcodes for building resume page Lots of components included e.g. accordion, tabs etc. Includes powerful theme options panel Localization ready HTML5 / CSS3 Valid and cross browser compatible Smooth CSS3 transitions 400+ icons of fontawesome Comes with custom contact form Full blown blog support Fully layered PSD is included Welll documented Dedicated support (free demo setup)
