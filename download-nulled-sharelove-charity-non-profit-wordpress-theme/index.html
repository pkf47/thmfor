Download theme

ShareLove is a responsive nonprofit WordPress theme that has all it takes if you are looking for a website that stands out from the crowd.

This WordPress Theme is fully equipped with the (Theme Options Panel) where there are all options available for each section which could be customized very easily. Each of the options has a detail explanation about them and how to use them in the documentation provided.

Theme Option panel also provide you the facility to change the color scheme and layout. You can also enable or disable sidebar for blog page.

Sharelove theme provides you with four extra post types, Testimonial’s. These All post types has some extra fields. This theme includes a Short-code gallery where you can use short code easily in post or page as this short code will be automatically generated on gallery created.

The theme includes a page builder to make page template by drag and drop a lot of blocks for use has been added. Each section that can be modified from theme blocks panel is labeled with same name.

There are extra fields that are available for the Event custom post type.

We have provided a well documentation by which user will find it very easy to use this theme.
