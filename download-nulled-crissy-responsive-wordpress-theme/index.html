Download nulled Crissy - Responsive WordPress Theme


Crissy is a stylish, carefully handcrafted and fully responsive WordPress blog and shop theme. Ideal for writers, story tellers or just casual bloggers, though it can serve just as well as an image portfolio.

Download theme

Overview

WordPress 4.1 ready Fully responsive, fluid layout Masonry grid layout Built with HTML5 and CSS 3 Infinite scroll posts loading Post formats: standard, aside, quote, image, gallery, video, audio and link Beautiful, large typography, with more than 600 Google fonts to choose from 23 different social media icons ( display them in header, footer or both) FontAwesome icons Built in lightbox for displaying pictures Styled Google map with animated marker Drop down menu support Threaded comments support Widget area inside the footer PSD mockup included Translation ready (.po file included) Documented Theme Customizer: - See your changes in real time Upload your logo image and favicon Set entire grid block as a link, or just headings and featured images Change fonts, fonts sizes, colors, and set your own background image Change the look of featured images (circle or rectangle) Hide/show page elements Choose navigation type (infinite scroll, previous/next page, or numbered) Change animation for appended posts when using infinite scroll Type custom CSS for fine tuning Configure Google map Set your social media profiles And much more… Page Templates: - Gallery template for quickly turning your native WP gallery into a full width, grid layout gallery Archive template for displaying: latest posts, monthly archive, categories, tags, authors, and post formats (you can choose what to show) About me template Google map template New: WooCommerce support: - Turn your site into a shop Includes Ajax dropdown menu cart (it can display number of items, amount, or both)
