Theme Description

This is a WordPress version of one of most popular One Page HTML templates!

Download theme

North is a 100% responsive and parallax one page WordPress theme. North is powered with Twitter Bootstrap 3.2 Framework. It is clean and professional theme perfect for agencies and creative studios. With the Visual Composer plugin and a powerful Theme Options Panel, it can be customized easily to suit your wishes.

Features Revolution and Layer Slider Included Visual Composer with Templatera Included Unique and Easy One Pager building system based on Page Builder (Visual Composer) Easy setup with Demo Content WooCommerce support! Bootstrap 3.1 HTML5 & CSS3 100% Responsive Powerful Theme Options Panel Awesome Theme Customizer Filterable, Isotope Portfolio and Multiple Usable Font Awesome Icons Clean Code Working Contact Form 12 Awesome Versions Light, Dark and Night Layouts Fullwidth and With Sidebar Blog Pages Browser Compatibility Easy to Edit Animations Google fonts included Shortcodes Smooth Scrolling Pricing Tables Detailed Documentation Parallax, One Page Design Full Screen Super Slides And so Much More… Theme Support:
