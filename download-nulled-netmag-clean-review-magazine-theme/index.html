NetMag is gorgeous, clean and minimalist responsive magazine/blog theme.

Import all posts, pages, menus, widgets, theme options and homepage layouts in one click! Original images will be replaced with placeholder to provide fast import.

Simple, Optimized and Fast

NetMag is simple, lightweight and fast Google PageSpeed score is 89-91 out of 100 (tested on Nginx server with basic file and browser cache configured). It has minified css and javascript, balanced image sizes, optimized structure for search engines, and even more! Images will be replaced with lightweight copies for mobile devices!

Responsive, Social and Retina-Ready

This theme will fit your large desktop screen, all way down to your smartphone – and of course any Tablet or iPad in between. Theme is optimised for your touchable and mobile device.

NetMag Theme has ultra-sharp Retina graphics.

Featured with nice pop-up social-share buttons and icons in the left panel for your social profiles.

Built-in Review System

NetMag Theme is ideal for reviewing any kind of content. Easy to use built-in review system allows you to create as many rating criteria as you wish. If you use the different review criteria then this value is calculated automatically in realtime. But you can override it easily by putting in your own score.

Drag & Drop Page Builder

Easy to use Page Builder offers endless possibilities of content creation and organisation. Use any of the 12 page block layouts to create your homepage or any other page.

Filter content in blocks by Category, Tag, Reviews, Featured, select Featured articles or Reviews by Category. Also you can use Offset option to pass over any number of posts for any post block and order post by Post ID, Author, Post title, Post slug, Date, Modified, Random or Comment count.

Unlimited Colors and Fonts

NetMag Theme includes backend color picker for easy customization. Also you can choose the font you want from Google Webfont Collection. There are more than 600 fonts available.

Gorgeous Layouts

You can use one of 4 beautiful layouts to create Magazine or Blog pages. Plus you can select Right Sidebar or Fullwidth variations for homepage, archives and any category, page or post.

Theme Installation

A common issue that can occur with users new to installing WordPress themes is a “Broken theme and/or stylesheets missing” error message being displayed when trying to upload or activate the theme. This error message does not mean that the theme you have purchased is broken, it simply means it has been uploaded incorrectly.

You should upload the installable netmag.zip file, not the whole package downloaded from Themeforest.

Unzip the final zip file downloaded from Themeforest, open it and find netmag.zip. This is the theme you should upload to your WordPress site.
