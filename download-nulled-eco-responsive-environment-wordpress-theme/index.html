ECO Responsive Environment WordPress Theme is a modern theme based on latest design trends in industry. Its main focus is Environment related project, Environment Protection Agencies, NGOs, Fundraising Services, Animal Care, Environment Friendly Product Stores.

A very well designed WordPress theme with a blog, multiple galleries, unlimited color schemes, events, Crowdfunding, WooCommerce powered store, Timeline, Layer slider and team pages. Theme is created by using Twitter Bootstrap, with a responsive design, a perfect fit for all devices.
