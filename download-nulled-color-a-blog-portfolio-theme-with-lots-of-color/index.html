About Color

Color is a colorful theme. Not only is it a beautiful blog theme, it is also a lovely portfolio theme. Our portfolio post type comes complete with featured image, slider gallery or video support. Color also has a colorful team post type and fully styled Easy Digital Downloads integration plus loads more!

Download theme

Each post, page or post type can be updated with an individual accent color, this gives each page across your website a unique look and feel – the accent changes the border, the backgrounds on archive pages, buttons, blockquotes, links and you’ll even see the pagination between posts keeps the accent color. Perfect for all you colorful bloggers out there as well as fans of Google Material Design.

If you’re not a fan of the grid blog posts, don’t worry we’ve got full width blog support (just go to Customize > Layout Settings on the demo to test it out).

Color Options
