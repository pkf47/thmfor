Download nulled Merit - Responsive WordPress Wedding Theme


Stop looking, this is the only wedding theme for WordPress you’ll ever need!

Please give a big round of applause for Merit, a really fascinating wedding theme for WordPress. Merit is easy to install and setup, you can also modify the colours and typography straight from our theme options, which is powered by the powerful ReduxFramework.

Download theme

Merit is simply amazing and easy to work with. The homepage is fully widgetized, which means you can control the homepage layout using simple drag and drop.

There are also more than 500 Google Web Fonts in the theme options waiting for you to choose, you can use them to customise your wedding site typography. You can also modify the colours easily using built-in colour picker.

This WordPress wedding theme also comes with AJAX RSVP form, testimonials, gallery, history, events, countdown timer, slideshow and other fascinating features.

Fyi, the word Merit is an Indonesian slank word taken from the word “married”.
