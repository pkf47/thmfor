Full list of Features

Ultra Responsive Nixon is 100% responsive, each and every element including the awesome slider are fully responsive. Responsive Valid HTML5 and CSS3, Cross browser compatible Visual Page Builder Literally within minutes you can build complex layouts with drag and drop interface without touching a single line of code. Neat and clean design Clean, Modern Design can be used for any type of website Infinite posibilites With the advanced styling customization features, you can create endless possibilities by changing navigation styles, backgrounds for each section with colors, images or patterns. 16 Predefined Color Styles We’ve included 16 ready-made skins in different color schemes for different purposes. They can be a great starting point for your unique custom design. Modularity Nixon is like one big puzzle – all elements of the site are designed highly modularly so you can build webpage and customize it according to your needs and taste. Clean Code The code is well written using the best resources available and also all the files are well commented to make your work much easier. Sticky Navigation Have you ever been at the bottom of a webpage and wanted to go to another page, but didn’t want to scroll all the way back up to the top to select a menu item? With Solido’s sticky header, you no longer have to do that. Your main menu and logo will always be at the top of the screen, automatically following when you scroll. Filterable portfolio (Ajax masonry portfolio) Retina ready We have included retina display optimizations for all the main graphics used in the theme, so they look sharp on retina displays. Parallax Sections You can easily add 3D looking parallax backgrounds to every page sections. 4 Header versions You can use 4 different header layouts to best suit your site focus (social, ads, logo) and, on top of this, you can mix in 2 color schemes (light and dark). Documentation Our aim was to make your work with template easier so we created detailed documentation where you can find all the tutorials for working with Solido. Expanding projects area (via AJAX) Includes a slider and HTML content. It also supports media like youtube, vimeo, etc. Browser Compatibility Works and looks good on all major browsers, tablets and phones
