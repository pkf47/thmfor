Other Notable Features Coded with Support for One Page functionality () Responsive layout with Retina graphics icons Internet Explorer 9 Compatibility

VIBES was tested on IE9 and it works fine, except for one notable issue with body/header backgrounds. As this browser doesn’t support CSS gradient technique, which is very crucial for achieving unique look of this theme, I decided to display both body and header backgrounds in solid color. That means no background image in IE9. Please take a look at to see how it looks in IE9. As IE9 is becoming more and more marginal each month, this shouldn’t be an issue for most users, but I thought that it is fair to let you know about this.
