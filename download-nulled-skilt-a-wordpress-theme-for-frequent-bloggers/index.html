Thanks for checking out Skilt!

Skilt is a stylish tumblog-style theme for WordPress, aiming to impress you readers with gorgeous background elements and a minimalistic yet efficient approach in design, functionality and coding.

Download theme

Fully compatible with the latest WordPress 3.9, Skilt is so simple to use that it does not come with a theme options panel. Instead it makes use of the built in Theme Customizer, which provides all the options you need, and lets you preview the changes as you customize on the fly.

Fully Responsive and Retina Ready — Skilt looks nice on all devices, be it phone or tablets, it will adopt to any screen size and makes things look beautiful on retina displays.

SEO Optimized — Also coded by following best SEO practices and it is 100% compatible with industry-leading SEO plugins like “All on One SEO Pack” and “SEO by Yoast”.
