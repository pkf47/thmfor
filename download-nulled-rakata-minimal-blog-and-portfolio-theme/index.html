Description

Rakata is a clean and minimal blog and portfolio theme. The layout is designed specifically to make sure that the focus is on your content.

Download theme

General Features

Responsive design. Retina ready. Beautiful header block with intro text. Custom header background per page/post. Lots of shortcodes to enrich your content. Social icons for authors on profile page. Hidden sidebar. Localisation support with .po and .mo files. Five custom widgets (Author List, Dribbble Shots, Portfolio List, Single Image, and Social links). One or multi-column portfolio layout with optional sorting feature. Extensive Documentation.
