Nice description

Sequoia is Wordpress theme mainly designed for usage with WooCommerce plugin powered e-commerce site. With main, parent theme there are additional 3 child themes, for starting with specific site or e-shop category – fashion, food or handmade products. Child themes also include import file with all the assets from themes demos. Off course, with child themes and all available options, there are no limits what kind of sites can be created with Sequoia.

Also, theme is powered with Slider Revolution, Aqua Page Builder plugin (heavily adapted and improved with additional features), compatible with Yith Woocommerce Wishlist, Yith Woocommerce Product Ajax serach, WP MailChimp Lite, WPML plugins
