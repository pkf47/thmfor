Luxury Villa is a stilish and very flexible property showcase wordpress theme designed to help you promote your villa, apartment or any kind of property/properties you own. This fullscreen theme features 5 different homepages, 4 different property layouts, 2 different areas page, a drag and drop page buider (Visual Composer), is retina ready and provides infinite color variations.

Take your time and see bellow some of the it’s coolest features:

All features

5 homepage layouts - 4 property layouts - 2 areas layouts - Gallery page with subgallery feature Blog page with masonry option Contact page with map and contact form Reservation page Visual Composer page builder ($28 value) Quick reservation form in header Ajax controled forms Unlimited Colors (Customize fonts, colors, backgrounds) Typography control (System fonts and Google fonts) Automatic Image Resizing Left/Right sidebar options Fullscreen or 1170 page content option HTML (Valid) & CSS3 animations Responsive design (Mobile, Tablet and Desktops supported) Extensive theme options, with the help of Customizer Demo content available (with dummy images, xml import file) – preview images not included Import/export feature to backup your theme settings. PSDs available Child theme support SEO optimized Custom widgets Extensive documentation Translation Readywith a .po file or WPML Cross browser compatible
