Haze – One Page Retina Wordpress Theme

Haze is a responsive and retina ready One Page Wordpress theme. It’s suitable for any type of business, built for any needs and perfect for YOU, no matter if you use it as a photographer, web design agency, nonprofit organization or even as a music producer.

Download theme

We offer you a pixel-perfect design powered with the help of Visual Composer which gives you the power to create the page in no-time. Also, the admin panel of the theme is built on Redux Framework and it allows you to customize up to 4 colors. This means you can be very creative and create from Haze a unique template perfect for your needs.

Some of Haze’s awesome features

One Page Template – This theme was build with the user in mind. That’s why we’ve created a strong one page theme that will satisfy all your needs! Custom Post Types – Haze has ‘under the hood’ 3 custom post types that helps you create portfolios, testimonials and also teams with team members. 4 Custom Colors – With Haze, you can customize up to 4 colors from the backend. Visual Composer Page-Builder – You can build pages in no-time with the power of Visual Composer that is full of custom shortcodes made special for Haze. Redux Framework – The backend of Haze is built with Redux Framework, the most powerful framework on this moment. Revolution Slider($19 value) – With revolution slider you can easily create, edit or delete beautiful slides with nice animations and captions. Parallax Backgrounds – You can add very easy a parallax background with Visual Composer. WPML ( WordPress Multilingual) -Full Localisation Support (contains .po/.mo files) 800+ icons – Haze uses Font Awesome and also IcoMoon for nice retina ready icons. Custom Google Map – You can use the custom map from Google Maps very easy, also from Visual Composer. Special Blog Pages – We’ve created a special template for the blog, and also a lot of special options that will help you filter the blog content as you want. Retina Ready – Haze was built on the latest standards and that means also Retina Ready. Fully Responsive – Clubix is a fully responsive that works on all types of screens and displays. Online Documentation – Everything is open for our customers. We don’t hide anything, that’s why we’ve shared the documentation online and you can see it . Free Updates & Support – We love and respect our customers – that’s why we offer update & support for free.
