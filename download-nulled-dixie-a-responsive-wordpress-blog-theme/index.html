Welcome to Dixie Blog Theme
Dixie is a modern WordPress Theme aimed to deliver the best writing and reading experiences for both the bloggers and readers. Dixie is also designed to be straightforward to the main content, without being distracted with other less important components.

Download theme

Core Features

Scrolling Page Background Transition
Insert a nice background image into the page to match the content’s context. When you scroll down reading, the page background will automatically change the image!

Styled WYSIWYG
Edit your post in the WYSIWYG editor with real typography and colors you have set in the Theme Options. You will know for sure, what image size will look best, the perfect paragraph text length, etc.
