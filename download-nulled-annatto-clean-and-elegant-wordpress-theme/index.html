Annatto is powerful WordPress Theme designed by in a clean, minimalistic & elegant style. It is totally responsive so it adapts to any device it is viewed on. Annatto is very flexible, easy for customizing and very well documented. Main usage is aimed towards bloggers or photography, fashion and other creative related content web sites which primarily uses different types of media.

Download theme

Annatto is coded in pure HTML5, CSS3 and jQuery. It has a great and flexible responsive layout which scales from 1260px to 320px width. Annatto is built upon Bootstrap 3 framework (Mobile First) and it’s 100% Retina ready.
