About TuneUp

TuneUp is a wordpress theme for your personal blog. It is responsive and easy to use. It works well with all desktop and mobile devices. If you’re a writer, an author, a publisher and looking for a clean blogging solution then TuneUp is for you. All post formats are included. Features

Eye-catchy modern design Responsive Design & Retina ready HTML5, CSS3, jQuery powered Flexible Images & Videos Two blog layouts (with or without sidebar versions) Complete blog system (all post formats supported) Responsive video embed in various aspect ratios (youtube/vimeo …) Unlimited Sidebar Custom Widgets (Flickr, Instagram, Pinterest, Dribbble, Picasa, YouTube) Footer widget locations (3 columns) Beautifully styled elements (blockquote, code etc.) Optimal Reading Experience On Any Device 600+ Google Fonts Extensive Documentation & Support Forum Theme Customizer Support (Change fonts and link colors on live customizer)
