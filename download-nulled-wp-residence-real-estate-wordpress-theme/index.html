Download nulled WP Residence - Real Estate WordPress Theme


WP Residence is a Premium Real Estate theme designed for real estate independent agents and agencies, but you can also allow other people to submit their properties for free, for a fee or based on a monthly subscription. It has clean & user friendly design, and it features options like: Properties Advanced Search (with custom fields and dynamic search URL), Google Maps with property location markers (managed from admin), Ajax Filters for Properties list, user login & register with white labeled dashboard (no WordPress reference), front end property submit & edit, payment integration with PayPal and Stripe for CC Payments, Visual Composer Page Builder Plugin, dsIDXpress IDX Plugin support, WPML plugin support, custom real estate widgets, colors controls, various Advanced Search and header options, easy theme options, and many more…
