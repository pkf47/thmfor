Download nulled Nyobian - Blog And Portfolio Theme


Description

Nyobian is a responsive blog and portfolio WordPress theme. The layout is designed specifically to make sure that the focus is on your content.

Download theme

General Features

Responsive Design. Retina Ready. Localisation support with .po and .mo files. Extensive documentation. Portfolio post type. Some custom shortcodes to enrich your content.

Design Bits

Off-canvas navigation. This will clean your page and set your users’ focus on the content. Beautiful header with background image or video. Blog posts and portfolios can be displayed in vertical list or grid layout with masonry style. Portfolios index has an optional filter to choose between category. Put it on the left, right, or upper center of your portfolios.
