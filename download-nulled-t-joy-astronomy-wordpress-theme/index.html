Download nulled T.Joy - Astronomy WordPress Theme


Description

T.Joy is clean and multipurpose a responsive WordPress theme. It is designed in dark color scheme, inspired by space and astrology thematic. Perfect for a technology companies, agencies, blogs and other technology websites.

Download theme

But don’t feel restricted! It can be customized easily to suit your needs. T.Joy WordPress theme includes one of the greatest options panel – Redux Theme Options. You can easily change any theme color. The template comes with a lot of shortcodes so you can create any type of content you needed..

T.Joy comes with premium Revolution Slider, 5 homepage layouts, 2,3,4 columns portfolio layouts, 5 blog pages, Contacts page, bunch of Shortcodes and much much more.

Features 100% Responsive Built on Bootstrap 3 WordPress 4.1 ready Revolution Slider included ($19) 40+ shortcodes (with shortcodes generator) Theme Options Panel (Redux Framework Based) Unlimited Colors Theme Customizer support (a real-time appearance editor) Custom widgets (Flickr Feed, Latest Post, Social Links, Twitter Feed) Translation Ready (.pot file included) WMPL support Google Maps integration Working Contact Form (Contact Form 7 Plugin) Google Web Fonts Demo Content included (xml) LESS/CSS based theme Free Support and Updates
