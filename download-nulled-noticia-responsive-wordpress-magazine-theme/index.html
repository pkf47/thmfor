Download nulled Noticia - Responsive WordPress Magazine Theme


Noticia is an advanced retina-ready news and magazine for your WordPress site. Build your news site instantly using Noticia and feel how easy it is to customize it’s colors, typography and other settings.

Download theme

This magazine theme comes bundled with reviews & rating, sleek responsive layout, custom post types, banner ad management an other cool features. Create your own news site with ease using this fully featured magazine theme.

Noticia Magazine Theme Features

Easy to use yet a very powerfull theme options (powered by Redux Framework) Unlimited colors Responsive layout 3 custom post types (Slideshow, Gallery, Team) Custom Page Templates
There are 5 additional page templates that comes with Qaween theme, they are: Homepage, Gallery (3 Columns), Gallery (4 Columns), Team and Full-Width page templates. Custom Widgets
There are 17 custom widgets to help you easily setup the theme Sidebar options, you can choose to use left, right or left-right sidebar from each post / page. Tons of useful shortcodes (powered by Shortcode Ultimate) Child theme support bbPress basic suppport Google Analytics integration Custom logo & favicon prettyPhoto lightbox Filterable gallery Translation ready and RTL support Enable/Disable share buttons on each post / page WordPress theme Customizer support
