Description

Fayce is the perfect, clean & simple ajax loading showcase theme for your projects to shine. It features a custom homepage layout with portfolio items and ajax powered loading of the portfolio item’s content.

Download theme

Features

General

Custom homepage portfolio Ajax loading for portfolio items Intro header with custom text Outro footer with custom text and/or button Full blog layout Clean design Responsive Translation ready

Wordpress customizer

change the colors disable parallax text in header disable parallax background in header enter social icon links enter header intro and text enter homepage footer outro, text & button display footer widgets on homepage disable ajax loading set standard text for ajax buttons Upload a favicon Upload a logo image

Custom portfolio items

change background color (if not set it will revert back to odd/even colors set in the customize – colors section) change background image change color of header text change color of normal text align media left, right or centered (if not set it will revert back to the odd right/even left alignment) choose to display stacked images choose to display a flexslider display video

Shortcodes plugin

The latest playneshortcodes plugin has been included in the theme download. Use it to easily make detailed pages or posts. Just install it from inside your admin panel and you’ll be able to add shortcodes from a dropdown menu inside your post editor.
