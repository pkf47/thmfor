Theme Highlights

Responsive & Retina Ready

This theme is optimized for a Responsive viewing experience.

Front-End

Multiple Slider Types

We have built-in 3 different sliders for you to choose from, with cool looking animations, while still being frugal with browser resources. They work great, look amazing and are easy to customize thanks to our Slider module.

Icon Fonts Integration

Icon Font Integration means that you will find lots of icons included in a single font file, and replacing them is as simple as editing a text by switching characters. Also, they are all vector graphics, so they scale up to any size without quality loss.

Built-in Weather Widget

Any hotel website needs a proper weather widget to show the local conditions of the destination. So we’ve built an easy-to-use widget where you just need to enter the location’s name and you’re set.

Mega Menu

MegaMenu is a user-friendly, highly customizable responsive drop down menu. Our implementation makes it easy to create highly customized and creative mega menu configurations.

Built in Page Layouts

Besides the regular default blog pages, Paradise Cove comes with a bunch of built in page layouts for you to use. From Rooms and Services to Locations and Amenities.

Built with Bootstrap

Bootstrap makes front-end web development faster and easier. It’s made for folks of all skill levels, devices of all shapes, and projects of all sizes. Also, Bootstrap is open-source and maintained on GitHub.

Back-End

Custom Posts

Custom Post Types help us bring more flexibility into WordPress and we use these type of posts to create different custom pages. This enable us to create easy to use options and features for the theme

Reservations

Use this powerful feature to receive reservation requests directly in your WordPress admin. You can also approve or reject requests or even communicate with the users using this module.

Advanced SEO Options, Built-in!

Search Engine Optimization (SEO) is vital for any website that wants to have as many visitors as possible, so we have built-in a great SEO module that offers you complete control & takes care of everything!

Powerful Contact Form Module

The theme comes with a contact form module that will enable you to create any contact form you might want or need. All this only a couple of clicks away.

Translation Ready

All our WordPress themes are translation ready, so they support free or premium plugins, like qTranslate and WPML. Using these tools you can translate everything: pages, posts, menus, taxonomy, etc.

Chock-full of Shortcodes

A shortcode is a WordPress-specific code that lets you do nifty things with very little effort. Shortcodes can create objects that would normally require lots of complicated code in just one line.

Unlimited Sidebars

A useful module when, for example, you want to add more than one sidebar to a page, or different sidebars on different pages. This brings yet another layer of customization freedom to your website.
