Download nulled Ephemera–Showcase EDD Shop Blog In Victorian Style


“Ephemera” is carefully crafted WordPress.org theme designed in Victorian style, perfect for showcasing and selling vintage and/or vintage style stuff.

Download theme

Read more about it .

Theme structure

The theme consists of 3 major blocks: showcase, shop and blog. Showcase block may serve well for any separate off the blog specific group of posts which are suitable for portfolio, showcase or collection of favourite items. Shop is built upon EDD free plugin. Blog is… well, it is a blog. What it does suit to?

The theme suits perfectly for selling and showcasing vintage style digital art, scarpbooking, vintage fonts, graphics, fashion and interior design mockups, stationary graphics, icon designs and similar things.

What is included

Victorian style collage and related graphics (designed by me) Vintage style social icons (created by ) Custom widgets and small plugins (written by me) Three (free version) fonts of Nexa family (by ) Help file in PDF format XML file for importing demo content

Requirements

There are some, very few and very little requirements to run this theme. You need to download, install and read its documentation. You need to create for at lease one post in showcase. You need to crop images for showcase proportionally–does not matter which size the images are but keep same proportion of all the images in this block posts. Same refers to Shop images (it is not obligatory but in case you want to keep that layout accurate and beautiful…). There is no way to skip using Showcase block on Home page and / or delete it from the Home page unless you are very comfortable with php and WP coding. You must be familiar with WordPress dashboard interface–what means you should know how to write and publish posts, drag widgets, insert images, add categories and similar things.

Features

Page templates: Home, Contact, Showcase, Full-Width, With Sidebar, With Sidebar And Comment Form, Shop Pages Slider for Home page Draggable latest posts for Home page Masonry layout for Showcase entrance page Like and Share for Showcase page Infinitely rotating small graphics Fully styled Shop with zoomed images Standalone search for Shop Boxes for extra info for Shop Social profile widget (icons included) Working php form for Contact Modified WordPress Gallery plugin with prettyPhoto lightbox Very specially styled sticky post for blog font@face fonts Nexa (included) Translation ready Accessibility ready Suitable for desktops (from 2560px down to 1280px) Options via WordPress Customiser Custom gravatar supported Galaxy tablet screenshots:

What is not

No page builders No third party plugins No bootstrap or any other framework No annoying animations in huge amounts No parallax effect No colour picker (change colours via CSS only) No custom shortcodes No mega menu (1 level submenu only)
