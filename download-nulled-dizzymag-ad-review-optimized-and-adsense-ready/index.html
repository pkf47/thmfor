Theme Description

Amazing theme for creative entrepreneurs, all you need for set up a successful online magazine and even more in the one theme, Dizzy Ezine comes with plenty of advertising placements and responsive AdSense, furthermore it is featured sortable portfolio and support WooCommerce.

Dizzy Ezine is a professional advertising and AdSense optimized magazine WordPress theme. The user friendly theme is ideal for creative entrepreneurs who seek to maximise monetization of their web projects. The Dizzy theme has been specifically designed for unlimited advertising conversion with plenty of advertising placements in hot spots. What’s more, the theme comes with built-in solution for making AdSense units responsive so it is ensuring the presence of ads regardless of the device being used. Because SEO optimisation is priceless for every website, the Dizzy theme is built with SEO best practice which causes prompt and proper indexing in search engines and as result brings thousands of visitors to your website.

The theme features a sortable portfolio so you can show your visitors what it is you do and how good you are in it. More over thanks of WooCommerce compatibility you can even sell your creativity directly from your website. One more noticeable feature is authors page which helps to present information about writers and readers can find articles by certain author.

Since the theme is fully compatible with WP Review plugin you will be able to gain more quality feedback from your visitors. WP Review plugin allows to create reviews, using stars, percentage or point rating so you will be able to write professional review for any goods and services. That’s really good option for those how wants succeed at the affiliate marketing. Some of the other notable features are boxed or full width layouts, light and dark skins, mobile responsive layout, WooCommerce, BuddyPress, and bbPress compatibility, translation ready, font icons, unlimited sidebars, custom styling options and more.

The theme comes with incredibly clean and valid PHP-code and corresponds to the modern WordPress standards. The HTML-code is properly written with respect to W3C-standards. JS-code is well written too.
