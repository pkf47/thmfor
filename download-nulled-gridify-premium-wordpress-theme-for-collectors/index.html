Download nulled Gridify - Premium WordPress Theme for Collectors


Gridify is grid-based WordPress theme for bloggers and creatives. Gridify is all about collecting and sharing with readers. I believe that i created simple yet powerful theme that will help You start own website in minutes, with all needed pieces in one place

Download theme

Since it is responsive, theme will adapt to different screen sizes which will make your website be compatible with any device such as smart phones, tablets or desktop computers. Additionally Gridify is retina ready so it will look sharp on these new fancy monitors.

All options in customizer – no confusing options and guessing what’s gonna happen. Preview all changes before publishing them.

Custom logo – use text or upload own image.

Custom background – builded-in support for custom backgrounds, pick color or image.

Layout – pick one of four available layouts modes.

Font Awesome – gives you vector icons that can instantly be customized. One font and 369 scalable & retina ready icons on board.

Built in sharing – built beautifully styled sharing right into the theme, share your content with style.

Built in ads – i got it! Fully merged ads system, including ads in AJAX loaded content.

Shortcodes – install plugin that comes with theme to get access to shortcodes. Buttons, tabs, tours, separators, columns and sections. Keep plugin when switching themes to move with short codes.

Widgets – theme have six defined sidebars for full control over widgets and where display them. Four widgets come with theme: twitter feed, flickr feed, dribbble shots and ads widget.

Search engine optimization – theme is builded with best practices and set of options that allow customize title, description, keywords and behavior of robots.

Review – review system is builded-in! Add infinite number of criteria and ratings, easy and intuitive interface.

Related posts – when they start to read, related posts will allow readers in easy navigation to similar content.

Advanced Custom Fields – powerful plugin that manage meta fields in theme.

Contact template – template with builded-in contact form, few clicks and readers can contact You directly.

AJAX – content loaded with AJAX, sexy and simple solution for infinite loading.

Social links in footer – share social links easy way, just paste them in right place.

Video & audio – use shared media from services like YouTube and SoundCloud or create player from own files.

All post formats – all WordPress post formats are supported: standard, image, gallery, video, audio, quote, link, status and aside.

Dummy content – with theme download dummy content, this way You can hie easier start with Simplify.

Custom CSS – special field in customizer allow to add own CSS to website just in few clicks.

Translation – inside theme files located languages folder, there are .po and .mo files ready for translation.

Support – dedicated support forum, i answer personally so you’re guaranteed a great & quick answer to your theme usage or setup question.
