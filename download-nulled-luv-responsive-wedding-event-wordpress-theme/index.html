Download nulled LUV - Responsive Wedding Event WordPress Theme


LUV is a nice and responsive one page WordPress theme for wedding, engagement or other events purpose. It comes with a simple yet elegant parallax design. Running with fully powered theme options by PlutonThemes, and very nice page builder that allows you to control every content that you want to show in every page.

Download theme

LUV is developed by Redux Framework which is well known framework on themeforest and top hot trend plugins such as Visual Comopser for building layout from frontend.

Key Features

Full Responsive Powerfull Theme Option Advanced Typography Custom Post Types One Click Install Sample Data Demo Support WPML Drag & Drop Page Builder CSS3 Animation Smooth Scroll Section: About Couple, Where and When, Bridesmaid, Groomsmen, Gallery, Blog, Registry, RSVP 2 Gallery Versions 2 Blog Versions Included Color: Pink, Purple and Brown
