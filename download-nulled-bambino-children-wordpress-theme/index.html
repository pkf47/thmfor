Theme Highlights

Install and get started in minutes – The theme only relies on pages and posts. There are no page builders or custom post types, which means you can probably import or export your existing content with just a few clicks. You can set globaloptions using the Theme Customizer and some page specific options using custom meta boxes.

Choose layouts that work for you – Publish your content using pages and create photo galleries, then use the Grid page template to showcase those pages using a grid or masonry layout. The theme will style everything so that it looks great on any device. Use the Theme Customizer to upload your logo, make different tweaks and choose a blog layout that works for you.

Create menus for easy navigation – It’s important to have a navigation that’s intuitive and allows users easy access to your content. Bambino can build menus from your existing structure (pages, categories, tags), or you can take matters into your own hands and create custom menus for the main, mobile and blog navigation.

Built on a strong foundation – Bambino is based on , a starter theme created by the fine folks at , creators of WordPress. So you know you’ll be using valid, secure, SEO optimized code. Top that with CSS that not only loads quickly, but also focuses on lots of white space, great typography and pixel perfect design elements.
