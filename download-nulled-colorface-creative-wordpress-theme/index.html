Download nulled Colorface - Creative Wordpress Theme


Download theme

Colorface is clean and responsive wordpress theme. Based on grid layout, ajax and animated transitions, the theme is made specially for who want to build a creative portfolio website.

Features

WP3.9 ready Fully Responsive WordPress Theme and Theme-Options Manager AJAX Support Animated Transitions Grid and responsive design layout Unlimited colors Google fonts intergration Easy to custom 2 kinds of gallery and support to embed video Easy to upload and sort images Post types support slideshow,embedded video, soundcloud, selft-hosted video and audio, quote Unlimited sidebars Customized widgets for portfolio, flickr, pinterest, video, etc. Shortcode editor button Font awesome intergration Ready for translation with .pot file Well document with tutorial video XML demo included And much more…
