Savant is a Clean, Retina Ready, Ultra Responsive and Highly Customizable WordPress Theme. You can create lots of different Business, Photography, Portfolio and Multi Purpose sites due to the ultimate customization power provided by the theme. It has been carefully crafted and obsessively improved to focus on meeting your expectations when it comes to content creation and productivity.

You have at your disposal lots of different tools for content and layout creation. Adjust the shadows, borders, colors, HTML5 Video and Parallax Backgrounds and many different stunning effects, without having to write a single line of code, all done from the theme’s admin interface. You do not need to have any advanced knowledge of WordPress to use the theme, it all feels intuitive enough that you’ll just find it natural to work with the theme.

We have developed the tools with you in mind, making sure there are no gotchas. We crafted them and went through a beta testing process to ensure every aspect of the theme is where it needs to be. Savant feels more like a complete application suite, providing several different tools with optimized workflows to help you create amazing sites very fast and with minimal effort.

Let the features and demo do the talking. Enjoy!
