Details & Features

Olida is a super clean responsive one page WordPress theme packed with features! Perfect for creatives, professionals and personal websites. Build your content with the powerful Visual Composer page builder plugin, create amazing slideshows with the popular Revolution Slider. We’ve included 5 pre-defined colors so you can quickly choose a style (you can also setup your own custom color). A full multi-post format blog allows you to create news stories with ease.
