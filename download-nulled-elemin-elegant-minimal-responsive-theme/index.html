Download nulled Elemin - Elegant Minimal Responsive Theme


Elemin has great sense of typography, good white space balance, and minimal graphics. It is a piece of art crafted with modern technology — Google font, CSS3, Javascript and HTML5. WordPress post formats was incorporated which allows you to write about various post types such as photo, gallery, video, quote, etc. (very much like Tumblr). The design is completely fluid and responsive. This means the layout automatically adapts based on the user’s viewport. The layout never breaks no matter of what screen resolution it is being viewed on. It works on all desktop and most mobile devices such as iPhone, iPad, Android, and Blackberry. To see this in action, visit our and resize your browser window or check it with a mobile phone.

Themify Builder = Limitless Layout Possibilities

Using (comes with the theme), you can build an unlimited number of layouts by dragging and dropping modules of your choice. These layouts can include static content, like text, images, videos, ads, and dynamic content, like queried posts, contact forms, and shop products. Unlimited Magazine Styling with Customize Panel

It comes with a Customize panel that allows the customization of theme elements and options with live frontend previews. You can select over 600+ Google fonts, use the color picker with opacity support, upload background images, set padding/margins/borders, write custom CSS, and much more.

Fluid & Responsive Design

The layout automatically adapts to fit on any screen resolution on desktop and mobile devices (iPhone, iPad, Blackberry, and Android).

List View & Grid View Layouts

Elemin offers a default list view layout and multiple grid view layouts. All these layouts work on mobile devices as well.

Default Layout Options
