Duality – Multipage Singlepage WordPress Theme

Duality is modern and creative single page WordPress theme with additional inner pages for every custom post type, page templates with left and right sidebar and blog pages. Built with shortcodes and PAGE BUILDER. Theme has 4 color options.

NOTICE: This theme is single page theme but it can become multi-page theme using additional page templates and second menu for pages. Please take a look at theme’s preview and pages that exist as inner pages before purchase.
